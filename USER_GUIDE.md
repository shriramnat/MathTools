## Getting Started

Open the app in a web browser. The screen shows a grid of math problems, each on its own card. Children write their answers directly on the cards using a finger, stylus, or mouse.

The top bar at the top of the screen contains all settings and drawing tools.

## Practice Modes

### Free Practice

The default mode. An infinite scrolling grid of problems appears on screen. New problems are generated as you scroll down. There is no time limit and no scoring.

Use this mode for casual, pressure-free practice.

### Session Mode

1. **Select session size** â€” choose 10, 15, or 20 problems from the dropdown
2. **Tap â–¶ Start** â€” the session begins and a timer starts counting
3. **Solve the problems** â€” work through the fixed set of problems

- The problem count is fixed â€” no infinite scrolling

## Difficulty Levels

### Easy

- Addition problems avoid carrying (no digit-by-digit overflow)
- Subtraction problems avoid borrowing
- Multiplication uses a single-digit second operand
- Both operands have the same number of digits

Best for: beginners, building confidence.

### Medium

- Carrying and borrowing are allowed
- Multiplication allows up to 2 digits for the second operand
- Operands can have different digit counts

Best for: children comfortable with basic operations.

### Hard

- Full difficulty within the selected digit range
- Multiplication can use the full digit range for both operands
- Operands can have different digit counts

Best for: advanced practice and challenge.

## Guided Mode

Toggle the **Guides** checkbox to enable visual scaffolding on problem cards.

When enabled, each problem card shows:

- **Faint vertical column lines** below the numbers â€” helps children align digits by place value (ones, tens, hundreds, etc.)
- **A carry row** above the first number â€” provides space to write carried digits
- **A dotted answer baseline** below the separator line â€” guides where to write the answer

These guides do not interfere with drawing and can be turned on or off at any time.

## Tips for Parents

### Choosing the Right Settings

- **Start with 1 digit, Easy, Addition only** for young beginners (age 5â€“6)
- **Increase digits gradually** â€” move from 1 to 2 to 3 as your child gains confidence
- **Add operations one at a time** â€” master addition before adding subtraction, then multiplication
- **Use Guided Mode** for children learning column addition/subtraction â€” the place-value lines help with alignment

### Using Session Mode Effectively

- Sessions create focused practice windows with a set number of problems
- The timer helps track how long practice takes, but there is **no time pressure** â€” let children work at their own pace
- Use the check panel to review answers together after the child finishes
- Discuss incorrect answers as learning opportunities

### Drawing Tips

- On tablets, a **stylus** provides the best writing experience
- **Increase brush size** (slider in the toolbar) if your child's strokes are hard to see
- Use **different colors** to make practice more engaging
- The **undo button** (â†©) on each card lets a child fix mistakes without clearing everything

### Offline Use

MathCanvas works entirely offline. No internet connection is required, no data is sent to external servers, and no accounts are needed. All progress data stays on the device.
